# commoji

## Usage

Clone the repo
```bash
$ git clone git@github.com:matchai/commoji.git
```
`cd` to the repo: 
```bash
$ cd commoji
```
Install module globally: 
```bash
$ npm install -g commoji
```
Initialize commoji globally 
```bash
$ commoji --init
```

:tada: Commit as usual, commoji is now applied globally to all your repos
